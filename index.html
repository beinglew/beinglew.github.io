Project Scope

This project is being developed for Fall 2022 Senior Capstone.
Mileage will integrate Oracleâ€™s MySQL, using Bluehost servers. To bridge Swift, the main programming language in which the iOS application is written, to MySQL and Bluehost, the mobile application and server will interpret PHP scripts. Bluehost offers a GUI in which the project owner can create and change routes as well as perform essential human resources functions. Phase 1 will implement the database only for clients, or drivers. Phase 2 includes functional requirements for the project owner to access route assignment and HR through the mobile application itself. Database integration is a work-in-progress.
